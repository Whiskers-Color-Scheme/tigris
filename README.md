## 👷‍♂️ Install

## Store

- Go to Settings -> Style -> Store -> Apply

## Manually

- Download the desired theme from [releases](https://github.com/Whiskers-Color-Scheme/tigris/releases)
- Go to Settings -> Style -> Import. And select the theme you downloaded
